# Library Management System

This project is a simple **Library Management System** developed as an **academic project** during the 3rd year of IT Engineering.

The project demonstrates basic **web development** and **database connectivity** concepts.

## Features
- User interface built using HTML and CSS
- Database connectivity using a DB connector
- Basic library-related operations (data handling)

## Technologies Used
- HTML
- CSS
- Database (MySQL)
- DB Connector
- Eclipse IDE

## Project Type
Academic / Personal Project

## Purpose
This project was developed to understand:
- Frontend structure using HTML and CSS
- Database connectivity
- Integration between frontend and database

## How to Run
1. Open the project in Eclipse
2. Configure the database connection
3. Run the application on a local server
